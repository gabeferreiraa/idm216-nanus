Project Overview

For this group project, our development team was tasked with building a responsive web application for Nanu’s Hot Chicken, a food truck known for their spicy chicken offerings. The project ran from January to March 2025, and as one of the developers, I collaborated closely with UX researchers, UI designers, and my fellow developers to bring the app to life.

Development Objectives

Our main goals were to:

  • Create an intuitive interface that allowed users to browse the menu, customize orders, and easily check out.
  • Develop a fully responsive web app based on Figma designs, ensuring a seamless experience on both desktop and mobile.
  • Enhance customer satisfaction through user testing, gathering feedback to refine the app’s functionality and flow.

Technical Approach

The app was built primarily with PHP to handle back-end logic. We structured the project to handle dynamic menu data, order customizations, and efficient checkout processing.

Our CSS styling leaned heavily on bold fonts and a color palette reflecting Nanu’s branding, while using media queries to make sure our project was responsive for all screen sizes.

Challenges and Collaboration

One of the biggest challenges as a developer was integrating the PHP back-end with the front-end design while keeping the experience smooth and responsive. Close collaboration with the design team helped us align on functionality and make necessary adjustments quickly. Another challenge arose with version control, but with some research we were able to deploy successfully on each push.

Regular team check-ins, along with using a Kanban board for task management, allowed us to stay on track and support each other effectively throughout the process.

User Testing & Results

Our UX team conducted rounds of user testing focused on menu navigation and the ordering process. The feedback was overwhelmingly positive — users found the app intuitive and easy to navigate.

Based on testing insights, we made small adjustments to improve the ordering flow and mobile responsiveness, ensuring a consistent experience across devices.

Reflection

This project was a valuable experience in full-stack web development within a collaborative team environment. Working alongside designers, and researchers helped me better understand the importance of user experience when building out new features. It was rewarding to see how our combined efforts resulted in an app that felt simple, functional, and met all of our clients expectations.
